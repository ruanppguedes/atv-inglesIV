📖 About

Task Manager is a simple and efficient Web App that helps users organize their daily tasks. Designed for productivity enthusiasts, students, and professionals who value clean design and fast performance.

✨ Features

  • ✅ Create, edit, and delete tasks
  • 📱 Fully responsive design
  • 💾 Data stored in browser (localStorage)
  • ⚡ Fast load time with no external libraries

🛠️ Technologies

  • HTML5
  • CSS3
  • JavaScript (Vanilla)
  • Git & GitHub
