🚀 How to Contribute

  1. Fork the repository
  2. Create a new branch: git checkout -b feature-name
  3. Make your changes and commit: git commit -m "Add feature X"
  4. Push and open a Pull Request

📐 Code Style

  • Use semantic HTML
  • Follow CSS naming conventions (BEM if possible)
  • Keep JavaScript modular and clean

📌 Issues and Pull Requests

We follow a basic workflow for contributions. Check for open issues before creating new ones.
